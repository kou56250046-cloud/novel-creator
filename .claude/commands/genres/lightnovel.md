# ラノベ系ストーリー コンテンツ作成手順書

## 基本情報

| 項目 | 内容 |
|------|------|
| ジャンルID | `lightnovel` |
| ファイル名 | `lightnovel.md` |
| 対象読者 | 中学生〜大学生（13〜22歳） |
| 文字数目標 | 2500〜4000字 |
| 語り口 | 一人称または三人称（主人公視点中心） |

---

## ジャンル定義・コンセプト

歴史上の人物を「個性の強いキャラクター」として描き、ドラマチックな演出・熱い展開・感情の起伏で読ませるラノベ風物語。
「プロローグ→各話→エピローグ」の章立て構成で、各キャラクターの見せ場（名シーン）を作る。
テーマは「使命・覚悟・成長」。感情を大きく動かす「熱いシーン」を少なくとも2か所作る。

---

## 語り手・主人公の設定

- **主人公像**: 選ばれし運命を持つ者。強さとともに迷いや感情を持つ
- **視点**: 一人称（「俺」）または主人公に密着した三人称
- **口調**: 現代的かつ文学的。「——そうだ」「——いや、違う」の独白スタイル
- **感情表現**: 内面描写を多用。「胸が、震えた」「言葉が、出なかった」など体感描写

---

## 構成テンプレート

```
【プロローグ】(200〜300字)
  謎めいた一言・情景描写・予告的な一節
  「これはその始まりの物語だ」的な導入

【第一話〜第N話】(各500〜700字)
  各話に一人のキャラクターを割り当てる
  各話の構成：
    - 状況説明（100字以内）
    - キャラクターの課題・葛藤（200字）
    - 行動・決断（200字）
    - 名シーン・名台詞（100字）
  話数は原典の主要人物数に合わせる（3〜5話）

【エピローグ】(200〜300字)
  物語全体を俯瞰する締め
  主人公の成長・気づきを表現
  「——END」または「——了」で締める
```

---

## 文体・表現ルール

### 演出テクニック
- ダッシュ（——）を多用して余韻を作る（「決めた——」「静かだった——」）
- 体言止めで緊張感を高める（「静寂。」「三秒の、沈黙。」）
- 「\n\n」改行を意図的に使って間（ま）を演出する

### 名シーン・名台詞の作り方
- キャラクターの信念・覚悟を一言で言い切る台詞を作る
- 台詞の前後に短い情景描写を入れる（「海が赤く染まっていた。」）
- 返答なしの沈黙または短い行動で台詞に重みを持たせる

### キャラクター描写
- 外見は最小限（一つの特徴的な描写のみ）
- 内面・信念・行動パターンで個性を出す
- 脇役キャラの反応で主人公の異常さ・すごさを際立てる

### テンポ
- 戦闘・決断シーンは短い文を連打（「斬る。受ける。踏み込む。」）
- 感情的な独白シーンは長めの一文で流す
- 会話文は短く切る。長セリフは分割する

## 必須要素

- [ ] プロローグで物語の「予感」を植える
- [ ] 各話に一人のキャラクターの「見せ場」
- [ ] 少なくとも2か所の「熱い名シーン・名台詞」
- [ ] キャラクターの行動・決断前の内面葛藤描写
- [ ] エピローグで主人公の成長・気づきを表現
- [ ] 「——END」または「——了」で締める

---

## 禁止事項

- 淡々とした説明調（すべて感情・行動・場面で語る）
- キャラクターが完璧すぎる（葛藤・弱さを必ず持たせる）
- 歴史解説の羅列（物語の流れの中に自然に組み込む）
- 長い説明的な地の文（場面転換は一行の情景描写で行う）

## 冒頭文例

```
「——お前には、天命がある」

その声が聞こえたのは、夢の中だった。

白い光の中に立つ自分。見渡す限り続く雪原と、遠くにそびえる黒い山。

「天命……？」

「〇〇の地に生まれし者よ。お前はこの民の物語を知らなければならない」

目が覚めたとき、窓の外に朝日が射し込んでいた。

そして俺は——なぜか泣いていた。
```
