## プロローグ　気づいたら神の息子だった件について

気づいたら、俺は空の上にいた。

「……ここ、どこ」

眼下に広がるのは見渡す限りの大地。白い山が遠くに見える。空気が澄んでいて、鳥の声が遠くに聞こえる。

「ここは天界です、桓雄様」

声のする方向を見ると、全身白い衣をまとった男が三人、深々と頭を下げていた。

「俺、桓雄？」

「はい。天帝・桓因様のご子息、桓雄様でございます」

天帝の……息子？

脳内でステータスウィンドウが開いた。

```
【ステータス】
名前：桓雄（ファヌン）
種族：神
所持スキル：
  ・農業知識LV∞
  ・医術LV∞
  ・法律制定LV∞
  ・カリスマLV∞
部下：3000名
使命：韓の地の人類文明化
```

「……チートすぎる」

思わず声に出た。

---

## 第一章　まず部下3000人を引き連れて降りる件

「桓雄様、地上への降臨は今すぐ可能です。いつ出発なさいますか」

俺は一つ深呼吸して答えた。

「今すぐ行く。全員来い」

「全員、とは……3000名、全員でよろしいですか？」

「全員だ」

「……かしこまりました（この方、やっぱりスケールが違う……）」

白頭山の頂上に降り立つと、麓の村人たちが全員こちらを見上げていた。

「神さまだ！！」
「空から人が……3000人！？」
「軍隊か！？」

俺は一歩前に出て、最初の宣言をした。

「農業・医術・法律——全部教える。ついてこい」

村人は静まり返った。

そして三秒後——

「「「ついていきます！！！」」」

こうして俺の地上転生ライフが始まった。

---

## 第二章　熊が最強の弟子になった件

「神様に弟子入りしたいのですが」

洞窟の前に、でかい熊が立っていた。

「熊が弟子入り……」

「虎もいます」

横を見ると、確かに虎もいた。

俺はスキル【適性鑑定】を発動した。

```
【熊の適性】
忍耐力：S
精神力：S
粘り強さ：S
全体評価：最強クラスの弟子候補
```

```
【虎の適性】
瞬発力：S
攻撃力：S
継続力：D
全体評価：短距離走者タイプ
```

「……わかった。条件がある。21日間、にんにくと白カビ草だけ食べて、洞窟から出るな」

「承りました！！（熊、目に炎が宿る）」
「余裕っす！！（虎、謎の自信）」

10日後——

虎から念話が届いた。

**《もう無理っす。アイツの意志力が怖い》**

**《俺、外に出ます》**

「……だろうな」

21日後。洞窟から現れたのは、長い黒髪の美しい女性だった。

「桓雄様。約束を果たしました」

俺は思わず固まった。

「え……いや、まあ、そうだね、うん」

彼女は後に俺の伴侶となり、朝鮮最初の王・檀君の母となる。

熊だったのに、なんかいろいろ最強すぎる展開だった。

---

## 第三章　転生者、後継者を育てる件（世宗大王編）

時代は流れ、俺は次の転生先で目を覚ました。

今度は王様だった。

```
【ステータス】
名前：世宗（セジョン）大王
種族：人間（天才型）
所持スキル：
  ・言語学LV∞
  ・統治LV∞
  ・民への愛LV∞
  ・眼病（デバフ）
```

眼病がデバフとして刺さっているが、それでも天才スキルが全部∞だった。

課題が浮かんだ。

**〔クエスト〕民のための文字を作れ**
**報酬：国民全員が文字を読めるようになる**
**期限：4年**

「やるか」

4年後——ハングル完成。

「なんと、一日で覚えられる文字が……！！」
「農夫のおれでも読めるぞ！！」

民が沸いた。

老臣たちが「秩序が乱れます」と反対してきたが、俺は一言だけ言った。

「読める国民が、秩序を作る」

反論なし。完封勝利。

*〔クエスト完了〕ハングル普及 ✓*

## 第四章　最強の海将、転生する件（李舜臣編）

次の転生先：李舜臣（イ・スンシン）将軍。

```
【ステータス】
名前：李舜臣
種族：人間（武将型天才）
所持スキル：
  ・戦略LV∞
  ・海戦LV∞
  ・発明LV∞
  ・不屈の精神LV∞
  ・讒言耐性LV∞
```

讒言耐性まで∞なのが草すぎる。

俺は即座に設計を始めた。

「亀甲船。鉄の甲羅で覆った戦艦。矢は無効。火炎も無効。四方に砲台」

「将軍……それ作れますか？」

「作る」

完成した亀甲船の前で、副官が呆然としていた。

「これ……化け物では？」

「出撃する。13隻で行く」

「え？敵は300隻ですよ？」

「足りる」

13対300——結果は全勝。23戦全勝。

戦略スキル∞は、文字通り無双だった。

---

## 第五章　そして俺は見守るだけになった件（柳寛順編）

最後の転生先で、俺は初めて気づいた。

転生先は、16歳の少女・柳寛順（ユ・グァンスン）。

ステータスを確認しようとしたが——ウィンドウが開かなかった。

「……あれ」

代わりに、一行だけ文字が浮かんだ。

```
この章は、俺のスキルではなく
彼女の意志だけで進む
```

「……そういうことか」

少女は広場に立った。一人で。スキルも、チートも、神の加護も、何もなく。

「大韓独立万歳！！」

その声を聞いて、民が集まった。

捕まっても。牢に入れられても。叫び続けた。

18歳で息絶えた。

俺は何もできなかった。

スキルなんてなくても、一人の人間の叫びが歴史を動かすことがある——それを俺は初めて知った。

---

## エピローグ

最初の転生から何千年が経ったか。

俺は今も、この民を見守っている。

スキルも、チートも、今はもう使っていない。

ただ——

白い服を着て、今日も生きている人たちを。

それだけを、見ている。

「弘益人間」

天命とは、「強くあること」ではなかった。

「広く、人間に益すること」——それだけだった。
