# ドラマ コンテンツ作成手順書

## 基本情報

| 項目 | 内容 |
|------|------|
| ジャンルID | `drama` |
| ファイル名 | `drama.md` |
| 対象読者 | 若手社会人（20〜35歳） |
| 文字数目標 | 2500〜4000字 |
| 語り口 | 三人称・主人公に密着した視点 |

---

## ジャンル定義・コンセプト

現代の若手社会人を主人公に据え、歴史書を読む・歴史的場所を訪れるなどの「現在」を起点に、
歴史上の人物の物語を「鏡」として自分自身の生き方・仕事・人間関係を見つめ直す構成。
「歴史」と「現在の主人公」を交互に描くことで感情を積み上げ、最後に主人公が行動を起こす。
泣ける・刺さる・背中を押される、が感情の三軸。

---

## 主人公の設定

- **年齢・職業**: 20〜35歳の若手社会人（具体的な職種を設定する）
- **状況**: 何らかの行き詰まりを抱えている（仕事・人間関係・アイデンティティ）
- **歴史との接点**: 親の遺品・偶然手にした本・旅先など自然な形で歴史に触れる
- **変化**: 歴史人物の言葉や行動を読むことで、自分の問題への向き合い方が変わる

---

## 構成テンプレート

```
【第一幕：現在の主人公と歴史との出会い】(300〜400字)
  主人公の状況・行き詰まりを簡潔に描写
  歴史書・古文・遺品などとの偶然の出会い
  「何かが引っかかった」感覚を植える

【第二幕〜第N幕：歴史人物の物語と現在の交差】(各400〜500字)
  歴史的人物の場面を描写
  → 現在の主人公が「これは自分のことだ」と重ねる
  各幕末に主人公の内面変化を一行描く

【結幕：主人公の行動】(300〜400字)
  歴史から受け取ったものを自分の状況に活かす
  具体的な行動（電話する・謝る・決断するなど小さくてよい）
  感動的だが過剰でない締め（静かで余韻のある終わり方）
```

---

## 文体・表現ルール

### 現在パートの書き方
- 感情を「体感」で描写する（「胸が重い」「指が止まった」「目が熱くなった」）
- 主人公の心の声は短く（長い独白はしない）
- 具体的な場所・時間・モノを描写して現実感を出す

### 歴史パートの書き方
- 淡々とした事実描写（現在パートとのトーン差を作る）
- 歴史人物の台詞は短く、行動で性格を示す
- 「だが彼は〜した」「それでも〜した」の逆接を多用して意志の強さを表現

### 「交差」の作り方
- 歴史パートの描写の直後に、現在の主人公が「気づく」描写を入れる
- 明示的に「これは自分に似ている」と書かずに、行動・感情で示す
- 歴史の言葉を主人公が心の中でなぞる・メモに書き写す

### 締めの作り方
- 大きなカタルシスより「静かな一歩」
- 主人公が誰かに連絡する・謝る・決断するなど具体的な行動
- 「弘益人間」など原典の核心的な言葉を自然な形で置く
- 窓の外の光・朝の描写など情景で感情を締める

---

## 必須要素

- [ ] 冒頭に主人公の「行き詰まり」の状況
- [ ] 歴史との「偶然の出会い」
- [ ] 歴史人物の場面と現在の主人公の交差（最低2回）
- [ ] 末尾に主人公の具体的な行動（小さくてよい）
- [ ] 原典の核心的な言葉・テーマが自然に登場
- [ ] 過剰でない、静かな感動の余韻

---

## 禁止事項

- 主人公が急激に変わりすぎる（小さな変化の積み重ねにする）
- 歴史の教訓を主人公が「〜だと気づきました」と直接語る
- 歴史パートが長くなりすぎて現在パートが薄れる（交互に均等に）
- 大きな感動シーン・泣き崩れる描写（静かな感動を優先）

---

## 冒頭文例

```
（場所）の（状況）。

（主人公の名前）は、（具体的なモノ）の前に立ち止まった。

「……（独白・短く）」

（いつから・何が変わったのか、一文で状況を説明）

（主人公）は（モノ）を持ったまま、（場所）に座り込んだ。

外は静かだった。
```
