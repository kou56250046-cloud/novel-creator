# 人生ドラマ コンテンツ作成手順書

## 基本情報

| 項目 | 内容 |
|------|------|
| ジャンルID | `life-drama` |
| ファイル名 | `life-drama.md` |
| 対象読者 | 中年層（40〜60代） |
| 文字数目標 | 3000〜5000字 |
| 語り口 | 一人称・語り手は50代前後の人物 |

---

## ジャンル定義・コンセプト

人生の折り返し地点に立つ語り手が、歴史上の人物の言葉・行動を手がかりに「自分の人生の意味」を静かに問い直すエッセイ調の物語。
「人生を問い直す問い」「歴史人物との対話」「過去の後悔と和解」の三層構造で感情を積み上げる。
説教・自己啓発にならないよう、答えは出さずに「余韻で終わる」ことが最重要。

**核心原則（Truby＋プルースト式）：**
語り手は「変わらない」。問いと共に生きることを「受け入れる」アーク。これが中年読者の最も深い感情層に触れる。
感情の名前を呼ばない（「悲しかった」ではなく「コーヒーが冷めていくのをただ見ていた」）。

---

## 語り手の設定

- **年齢**: 50〜55歳（定年が見えてきた時期）
- **状況**: 仕事・家族・自分の価値に関して漠然とした問いを抱えている
- **きっかけ**: 歴史書・親の遺品・病院の待合室など静かな場所で歴史に触れる
- **口調**: 知的・内省的・穏やか。自分に問いかけながら語る
- **重要設定**: 語り手が「完全な答えを持っていない」ことを明示する（「正直に言えば、あの選択が正しかったとは今でも思えない」）

---

## フックの技法（冒頭で読者を引き込む）

1. **人生の転換点の「直後」から始める**：
   - 例：「53歳のとき、私は退職届を書いた。書いてから、なぜ書いたのかわからなくなった」
   - 行動を先に見せ、動機を後から探る。中年読者が即座に自分の経験を重ねる
2. **細部の観察を冒頭に置く（プルースト式）**：
   - 例：「コーヒーカップの底に残った澱を、私は毎朝ながめている。なぜかはわからない」
   - 生活の細部に宿る真実。中年読者が最も共鳴する冒頭パターン
3. **普遍的な問いを具体的な「if」で提示する**：
   - 例：「あの交差点で右に曲がっていたら、私の人生は違ったのか」
   - 抽象論を避け、具体的な場面から問いを出す

---

## ページをめくらせる仕掛け

- **読者の「気づき」を語り手より先行させる**：読者が「ああ、この人は本当は○○が怖いのだ」と気づくのを、語り手本人が気づくより早くする。この「前方認識」が読者を縛り付ける
- **感覚的記憶のトリガー（プルースト式）**：「あの頃の夏は、必ずセミの声がした。今、セミの声が聞こえると、私は20年前に戻る」——特定の音・匂い・味が時代を呼び起こす
- **「言えなかった言葉」を核心に置く**：「なぜあのとき、私は父に『ありがとう』と言えなかったのか」——中年読者の最も深い感情層
- **解決されない問いを意図的に残す**：答えを出さないことが誠実さになる

## 構成テンプレート

```
【序】(200〜300字)
  語り手がいつ・どこで・なぜ歴史書を開いたか（日常の一場面から自然に滲み出る）。
  「自分はいったい何のために生きてきたのか」という問いの提示——唐突に言わず、細部の観察から浮かび上がらせる。
  冒頭は「行動の直後」か「細部の観察」から。説明から始めない。

【一〜N（各歴史人物の章）】(各400〜600字)
  ①歴史人物のエピソードを「記録にはこうある。〜」形式で引用
  ②語り手が自分の半生と重ねる（「あのとき、私は〜した」）
  ③「私はどうだったか。」という自問（短い文で行を分けて問う）
  ④答えは出さない。問いが深まるだけ

【結び】(300〜400字)
  すべての歴史人物を経て、語り手が今感じることを静かに語る。
  「それで十分だ」「それだけでいい」系の穏やかな着地——「変わった」ではなく「受け入れた」アーク。
  答えを出さず、「問いと共に生きる」ことを受け入れる。
  小さな行動（茶を淹れる・窓を開けるなど）で締める。
  原典の核心的な言葉で締める。
```

## 文体・表現ルール

### 感情を名前で呼ばない（Show, Don't Tell の徹底）
- 「悲しかった」→「コーヒーが冷めていくのをただ見ていた」
- 「嬉しかった」→「その日の夕飯が、なぜかいつもより美味しかった」
- 感動的な場面ほど文を短くして余白を作る

### エッセイ調の書き方
- 語り手が過去を振り返るとき：「あのとき、私は〜した」「〜年前のことだ」
- 歴史を引用するとき：「記録にはこうある。〜」「〜と彼は言った。」
- 自問するとき：「私はどうだったか。」（短い文で行を分けて問う）

### 沈黙と間を文章で表現する
- 「彼は何も言わなかった。私も何も言わなかった。窓の外で、鳩が飛んでいった」
- 「何も起きない」瞬間の描写が深さを生む（長文で意識の流れを再現）

---

## 必須要素

- [ ] 冒頭が「行動の直後」か「細部の観察」から（説明から始めない）
- [ ] 語り手の「人生の問い」が自然な形で提示される
- [ ] 歴史人物のエピソードが最低3人分
- [ ] 各エピソード後に語り手の「自問」（短い文で）
- [ ] 読者が語り手より先に「気づく」仕掛けが1か所
- [ ] 「言えなかった言葉」または後悔の解析が1か所
- [ ] 答えを出さない結び
- [ ] 原典の核心的な言葉・価値観が自然に登場

---

## 禁止事項

- 感情の名前を呼ぶ（「悲しかった」「嬉しかった」は使わない）
- 「この話から学べることは〜」という直接的な教訓提示
- 語り手が急に明るくなる・問題が解決する
- 感情的な泣き崩れ・叫び
- 自己啓発書風のまとめ（「5つの教訓」など）
- 若者向けの言葉遣い・流行語

## 冒頭文例

```
コーヒーカップの底に残った澱を、私は毎朝ながめている。

なぜそうするのかは、自分でもわからない。

（歴史書・本）を初めて開いたのは、ちょうどその頃だった。

定年まであと三年。子供は独立した。妻は別の部屋で寝ている。
人生が「整理された」ように見えるのに、何かが胸の奥に引っかかったままだ。

「自分はいったい、何のために生きてきたのだろう」

問いは唐突に訪れた。
それはいつものように澱を見つめていた、ある朝のことだった。
```
