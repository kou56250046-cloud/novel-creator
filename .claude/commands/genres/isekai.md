# 異世界転生系 コンテンツ作成手順書

## 基本情報

| 項目 | 内容 |
|------|------|
| ジャンルID | `isekai` |
| ファイル名 | `isekai.md` |
| 対象読者 | 中学生〜大学生（13〜22歳） |
| 文字数目標 | 3000〜5000字 |
| 語り口 | 一人称（主人公：転生者） |

---

## ジャンル定義・コンセプト

歴史上の人物に転生した主人公が、各時代の問題を「メンタル最強・合理主義・自由奔放」なキャラクターとして乗り越えていく物語。
笑いは「周囲が戸惑うのに主人公だけ平然としている落差」から生まれる。
感動は「合理主義で動く主人公が初めて感情的になるシーン」に集約させる（全体で1か所）。
**ステータス表示・ゲーム的なUI演出は使わない。** 言葉で描写する。

---

## 主人公の設定（固定）

| 特性 | 具体的な表れ方 |
|------|-------------|
| メンタル最強 | 何が起きても動じない。「なるほど」「そうか」「わかった」で受け入れる |
| 合理主義者 | 行動には必ず理由がある。「必要だから〜する」「〜する理由がない」 |
| 自由奔放 | 慣習・常識・他人の反応を気にしない。やると決めたら即行動 |
| 感情の見せ場 | 全体でただ1回。「合理主義が黙る瞬間」を最終章に持ってくる |

---

## 構成テンプレート

```
【プロローグ】(200〜300字)
  気づいたら転生している状況説明
  主人公が3秒〜数行で現状を把握し受け入れる
  「やること、多いな」的な即断で物語が始まる

【転生先①〜④（各400〜700字）】
  各転生先に一人の歴史的人物を割り当てる
  構成：
    1. 状況提示（問題が明らか）
    2. 周囲の反応（戸惑い・恐れ・反対）
    3. 主人公の判断（簡潔・合理的）
    4. 行動・結果
    5. 周囲の驚きに対する主人公の反応（「それだけだ」「次の準備をしろ」など）

【感動の転生先（最終章・500〜700字）】
  主人公が「初めて何もできない」状況に置かれる
  合理主義では解決できない、感情・勇気・声だけが武器の場面
  主人公が初めて揺れる・黙る・泣く

【エピローグ】(200〜300字)
  全転生を経た主人公の変化
  「弘益人間」など原典の核心的な言葉で締める
  「——END」で締める
```

---

## 文体・表現ルール

### 主人公の台詞パターン
- 承認：「なるほど」「そうか」「わかった」
- 行動宣言：「〜する」「〜する。以上だ」
- 説明：「〜する理由がない」「必要だから〜した。それだけだ」
- 礼を断る：「礼はいらない」「俺には関係ない」
- 次を向く：「次の準備をしろ」「次に行く」

### 落差コメディの作り方
- 周囲が「！？」「え！？」で騒ぐのに、主人公は短文で返す
- 周囲の長いセリフに対して主人公は3〜5文字で返す
- 「恐れ」を問われたときに「何が」と逆に聞き返す

### 感動シーンの作り方（最終章）
- 「主人公の内心がはじめて長く語られる」展開にする
- 「俺には説明ができなかった」「数値で表せないものが動いた」など合理主義の崩壊を描写
- 主人公が黙る・立ち尽くす・泣くなど外的反応で感情を示す

### 禁止：ステータス表示
- 「STR: 9999」「LV∞」などのゲームUI風表現は使わない
- スキル・能力値を数字で示さない
- 代わりに「俺が設計すれば三日で完成する」など台詞・行動で示す

---

## 必須要素

- [ ] プロローグで転生を「3秒〜数行で受け入れる」描写
- [ ] 各転生先に「周囲の戸惑い」と「主人公の即断」の落差
- [ ] 転生先ごとに異なる歴史的事件・人物（最低3転生先）
- [ ] 最終章で「合理主義が黙る場面」（感動の1点集中）
- [ ] エピローグに主人公の変化・学び
- [ ] ステータス表示なし

---

## 禁止事項

- ゲームUI（ステータスウィンドウ・スキル欄・数値）
- 主人公が感情的になる場面を複数作る（最終章の1か所に集約）
- 周囲が納得して普通に受け入れる（常に「この人おかしい」反応を維持）
- 長い自己紹介・説明セリフ（主人公は短く端的に話す）

## 冒頭文例

```
気づいたら、空の上にいた。

眼下に広大な大地。遠くに白い山。清涼な空気。

「……なるほど」

俺は三秒で状況を把握し、受け入れた。

「〇〇様！ご無事でございますか！！」

白い衣の男たちが慌てて駆け寄ってくる。

「何人いる、お前たち」

「〜名、ご命令のお供を承っております！！」

「多いな。まあいい」

俺は地上を見下ろした。やること、多いな。

「降りるぞ。全員ついてこい」
```
