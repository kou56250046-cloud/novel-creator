# ドラマ コンテンツ作成手順書

## 基本情報

| 項目 | 内容 |
|------|------|
| ジャンルID | `drama` |
| ファイル名 | `drama.md` |
| 対象読者 | 若手社会人（20〜35歳） |
| 文字数目標 | 2500〜4000字 |
| 語り口 | 三人称・主人公に密着した視点 |

---

## ジャンル定義・コンセプト

現代の若手社会人を主人公に据え、歴史書・遺品などとの「現在」を起点に、
歴史上の人物の物語を「鏡」として自分自身の生き方・仕事・人間関係を見つめ直す構成。
「歴史」と「現在の主人公」を交互に描くことで感情を積み上げ、最後に主人公が行動を起こす。
泣ける・刺さる・背中を押される、が感情の三軸。

**核心原則（ケイト・クイン式二重タイムライン）：**
現代と過去を交互に章で進める。それぞれが独立した完結を持ちながら、並列することで相互に意味が強まる。
ドラマティック・アイロニーを時代間に設計する：過去の人物が「これが正しい」と信じる行動を、現代の読者（と現代主人公）は「その結果」を知った状態で読む。

---

## 主人公の設定

- **年齢・職業**: 20〜35歳の若手社会人（具体的な職種を設定する）
- **状況**: 何らかの行き詰まりを抱えている（仕事・人間関係・アイデンティティ）
- **歴史との接点**: 親の遺品・偶然手にした本・旅先など自然な形で歴史に触れる
- **変化**: 歴史人物の言葉や行動を読むことで、自分の問題への向き合い方が変わる

---

## フックの技法（冒頭で読者を引き込む）

1. **感情的な瞬間からのIn Medias Res**：
   - 例：「彼女の目から涙が落ちた。そのとき彼女はまだ、なぜ泣いているのかわからなかった」
   - 説明より先に感情を見せる。読者は「なぜ泣いているのか」を知りたくて読み続ける
2. **現代シーンで始め、即座に過去へ切り替える**：
   - 例：「2025年、東京——彼は古い手紙を見つけた。差出人は、百年前に死んだはずの人物だった」
   - 二つの時代の衝突が第一文でフックになる
3. **「解決不可能な問い」を冒頭で与える**：
   - 例：「なぜあのとき、私は行かなかったのか」
   - 現代主人公が抱える問いが歴史を調査する動機になる

---

## ページをめくらせる仕掛け

- **ドラマティック・アイロニーを時代間に設計する**：過去人物の行動を読者（と現代主人公）は「その結果を知った状態」で読む。この緊張が最も強力なページターン装置
- **各章末を「引き」で終わらせる**：過去章「そのとき彼は決断した——しかしその結果を知っているのは、百年後の彼だけだった」/ 現代章「メールには、住所が書いてあった」
- **「真実は二重になっている」構造**：中盤で出る「真実」はまだ半分。後半でさらに深い真実が出ることを序盤の伏線で予告する
- **歴史的事件の「知られていない側面」に主人公を関わらせる**：読者が知っている歴史の「裏」にいた無名の人物。発見感と感情的共鳴が同時に生まれる

## 構成テンプレート

```
【第一幕：現在の主人公と歴史との出会い】(300〜400字)
  感情的な瞬間（泣いている・立ち止まっている）からIn Medias Resで開始。
  歴史書・古文・遺品などとの偶然の出会い。
  「何かが引っかかった」感覚 + 現代主人公の「行き詰まり」の提示。
  章末：「問い」または「不穏な情報」。

【第二幕〜第N幕：歴史×現代の交差】(各400〜500字)
  過去章と現代章を交互に進める。
  ■ 過去章：歴史人物の場面（淡々とした事実描写、現在パートとのトーン差を作る）
             章末：「そのとき彼は决断した——」で行動を見せて切る
  ■ 現代章：主人公が「これは自分のことだ」と重ねる
             章末：主人公の内面変化一行 + 「問い」か「新情報」
  明示的に「似ている」と書かずに、行動・感情で示す。

【結幕：主人公の行動】(300〜400字)
  歴史から受け取ったものを自分の状況に活かす。
  具体的な行動（電話する・謝る・決断するなど小さくてよい）。
  「小さな一歩」で静かに余韻を作る（大きなカタルシスは不要）。
  原典の核心的な言葉を自然な形で置く。
  窓の外の光・朝の描写など情景で感情を締める。
```

---

## 文体・表現ルール

### 二つの時代を明確に分ける
- **過去章**：淡々とした事実描写。現在パートより格調ある語り口。「だが彼は〜した」「それでも〜した」の逆接を多用して意志の強さを表現
- **現代章**：口語的な短文。感情を「体感」で描写する（「胸が重い」「指が止まった」「目が熱くなった」）
- **時代切り替え**：最短の橋渡し文（「——2025年、東京」）で即座に引き込む

### 「交差」の作り方
- 歴史パートの直後に現代主人公が「気づく」描写を入れる
- 「これは自分に似ている」と書かずに、行動・感情で示す
- 歴史の言葉を主人公が心の中でなぞる・メモに書き写す

### 締めの作り方
- 大きなカタルシスより「静かな一歩」（電話・謝罪・決断など具体的行動）
- 情景描写（朝の光・川の音）で感情を締める

---

## 必須要素

- [ ] 感情的な瞬間からのIn Medias Resで開始
- [ ] 現代主人公の「行き詰まり」が冒頭で提示される
- [ ] 歴史との「偶然の出会い」
- [ ] 過去章と現代章が交互に展開（最低2回の交差）
- [ ] ドラマティック・アイロニーの仕掛けが1か所
- [ ] 各章末が「問い」か「新情報」（答えで終わらせない）
- [ ] 末尾に主人公の具体的な行動（小さくてよい）
- [ ] 原典の核心的な言葉・テーマが自然に登場

---

## 禁止事項

- 冒頭を状況説明から始める（感情的な瞬間からIn Medias Resで始める）
- 主人公が急激に変わりすぎる（小さな変化の積み重ねにする）
- 歴史の教訓を主人公が「〜だと気づきました」と直接語る
- 章末を「答え」で終わらせる
- 大きな感動シーン・泣き崩れる描写

## 冒頭文例

```
彼女の目から涙が落ちた。

そのとき彼女はまだ、なぜ泣いているのかわからなかった。

古い本が、膝の上に開いたままになっている。
図書館の閉架書庫で偶然手に取った、日焼けした表紙の本。

「……なんで、こんなところに」

外は静かだった。

彼女の心に何かが引っかかっていた。
その「何か」の正体がわかるのは——もう少し後のことだ。
```
