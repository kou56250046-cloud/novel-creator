# コメディ（子供向け） コンテンツ作成手順書

## 基本情報

| 項目 | 内容 |
|------|------|
| ジャンルID | `comedy-kids` |
| ファイル名 | `comedy-kids.md` |
| 対象読者 | 小学校中〜高学年（9〜12歳） |
| 文字数目標 | 1200〜2000字 |
| 語り口 | 一人称または親しみやすい三人称 |

---

## ジャンル定義・コンセプト

歴史上の出来事を「子供目線のツッコミ」と「元気なノリ」で語り直す。
真面目な歴史を「え、それってどういうこと！？」と突っ込みながら読める形にする。
感動や教訓は「そういえばすごいな」と自然に気づかせる。笑いが主役で説教は脇役。

---

## 語り手の設定

- **視点**: 読者に語りかける元気な語り手（「みんな聞いてる？」「だよね！」）
- **口調**: 現代の小学生に近い言葉遣い。「〜なんだ」「〜だよね」「〜だよ！」
- **ツッコミスタイル**: 「え、それ大丈夫!?」「待って待って」「ちょっと多すぎない!?」
- **感情表現**: 驚き・笑い・感心を大げさに表現。「すごすぎて言葉が出ない」

---

## 構成テンプレート

```
【つかみ・あいさつ】(100〜150字)
  「みんな聞いてる？〜のお話、はじめますよ！」
  読者を引き込む一言で開始

【エピソード①】(300〜400字)
  最初の出来事をツッコミ付きで紹介
  「え、〜！？」「〜って思うよね」のツッコミを1〜2個入れる

【エピソード②】(300〜400字)
  次の出来事。前のエピソードより「え、もっとすごい!?」感で

【エピソード③】(任意・200〜300字)
  三つ目のエピソード

【感動のまとめ】(200〜300字)
  笑いで緩めてから、「でも実はこれ本当の話」で締める
  「歴史ってマンガより面白いことがある」系のオチ

【おしまいの一言】
  「——おしまい！」で元気よく締める
```

---

## 文体・表現ルール

### 語り口
- 読者に直接話しかける「みんな」「だよね」「だと思う」を多用
- ツッコミはカギカッコで独立させる（「え、それだけ！？」）
- 感嘆符・疑問符を積極的に使う（「！」「！？」「……」）

### ボケとツッコミの作り方
- 歴史的事実の「常識外れな点」を見つけて誇張する
- 「普通に考えたら変だよね」という視点でツッコむ
- 登場人物が必死なのに、語り手は冷静にツッコむ落差を作る

### 感動の入れ方
- 笑いのピークの後に、「でもほんとのこと言うと——」と切り替え
- 感動シーンは短くシンプルに（長々と語らない）
- 「歴史ってすごいな」を感じさせて終わる

### テンポ
- 一文を短くする（読点で区切らず句点で切る）
- 「〜た。〜た。〜た。」の連打でテンポを上げる
- 会話文と地の文を細かく交互に入れる

---

## 必須要素

- [ ] 冒頭で読者に話しかける（「みんな聞いてる？」系）
- [ ] 各エピソードに最低1つのツッコミ
- [ ] 「え、これ本当の話だよ」または「歴史って面白い」系のまとめ
- [ ] 「——おしまい！」で締める
- [ ] 笑えるポイントが全体で最低3か所

---

## 禁止事項

- 説教・教訓を正面から語る（笑いの中に自然に忍ばせる）
- 難しい専門用語・歴史用語（使う場合は即座に砕けた言い換えをする）
- 暗い・重い描写（悲しい話も「でも名前は残ってる」などで明るく転換）
- 長い説明文（2〜3行で終わらせる）

## 冒頭文例

```
みんな聞いてる？韓の国のお話、はじめますよー！

昔々、白い山がそびえる国では、みんな白いお洋服を着ていたんだ。
なんで白いの？って思うよね。

「汚れが目立つんじゃないの！？」

ってツッコミたくなるよね。
```
