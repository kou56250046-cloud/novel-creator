# 昔話風 コンテンツ作成手順書

## 基本情報

| 項目 | 内容 |
|------|------|
| ジャンルID | `folktale` |
| ファイル名 | `folktale.md` |
| 対象読者 | 幼児〜小学校低学年、読み聞かせをする親 |
| 文字数目標 | 1500〜2500字 |
| 語り口 | 三人称・昔話語り |

---

## ジャンル定義・コンセプト

日本の昔話（桃太郎・浦島太郎など）の語り口を借りて、原典の内容を「むかしむかし」から始まる温かい語りで伝える。
難しい概念を排除し、繰り返しのリズムと情景描写で子供が聞いても楽しめる形にする。
「めでたし、めでたし」で締めくくることで、読後感を安定させる。

---

## 語り手の設定

- **視点**: 全知の語り手（「〜じゃったとさ」「〜おったんじゃな」）
- **口調**: 方言風（〜じゃ、〜おった、〜とさ、〜のう）
- **感情**: 温かく、穏やか。驚きも「おや」「まあ」程度
- **説明スタイル**: 難しい言葉は使わず、事実をやさしく語り直す

---

## 構成テンプレート

```
【導入】(200〜300字)
  むかしむかし、〜の国に〜がおった。
  舞台と主な登場人物を紹介する。
  世界観・雰囲気を一段落で確立する。

【出来事①】(400〜500字)
  最初の大きな出来事（建国・発明・戦いなど）
  「ある日」「そのとき」などで場面転換

【出来事②】(400〜500字)
  次の出来事。前の出来事と対比や連続性を持たせる

【出来事③】(任意・300〜400字)
  三段構成の場合は三つ目

【結び】(200〜300字)
  「こうして〜」でまとめ
  「——めでたし、めでたし。」で締める
```

---

## 文体・表現ルール

### 語尾・語り口
- 文末は「〜じゃった」「〜おった」「〜とさ」「〜んじゃな」を使う
- 現代語の「〜です」「〜ます」は使わない
- 直接話法は「〜と言われたそうな」「〜とおっしゃったとさ」

### 語彙
- ひらがな多用（「白い」→「しろい」、幼児向けの箇所は全ひらがな可）
- 固有名詞は初出時にルビ風に読み方を添える（例：桓雄（ファヌン））
- 擬音語・擬態語を積極的に使う（「ドスン」「フワリ」「キラキラ」）

### リズム
- 三回繰り返しのパターンを活用（「一度め、二度め、三度め」）
- 短い文と長い文を交互に置いてリズムを作る
- 段落の最後に「〜とさ」「〜じゃったとさ」で締める

### 章立て
- 各セクションに「## タイトル」見出しを使う
- 見出しは「〜の話」「〜のこと」などの昔話らしい名前にする

---

## 必須要素

- [ ] 「むかし、むかし」または「むかしむかし」で開始
- [ ] 舞台となる国・地名の情景描写（山・川・空など自然描写）
- [ ] 登場人物の人柄を一言で表す描写
- [ ] 困難・試練のシーン（乗り越える過程を描く）
- [ ] 教訓となる言葉・価値観（やさしく・さりげなく）
- [ ] 「——めでたし、めでたし。」で締める

---

## 禁止事項

- 難解な歴史用語・専門用語をそのまま使う
- 暗い・重い結末（悲劇は「でも民は覚えておった」などで昇華する）
- 長い説明文（体言止めや情景描写で代替する）
- 現代語・カタカナ語（「システム」「プロジェクト」など）

## 冒頭文例

```
むかし、むかし、大きな海のむこう、白い山のそびえる国のことじゃった。

その国の民は、雪のように白い服をこよなく愛しておった。
朝は天に手を合わせ、夜は星を眺めながら歌い、笑い、泣いて暮らしておったとさ。
```
