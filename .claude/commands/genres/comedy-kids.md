---

## ジャンル定義・コンセプト

歴史上の出来事を「子供目線のツッコミ」と「元気なノリ」で語り直す。
真面目な歴史を「え、それってどういうこと！？」と突っ込みながら読める形にする。
感動や教訓は「そういえばすごいな」と自然に気づかせる。笑いが主役で説教は脇役。

**核心原則：**
「1場面=1笑い」。各段落を1つの笑いポイントで終わらせ、読者が段落末尾を「報酬」として期待するよう設計する。

---

## 語り手の設定

- **視点**: 読者に語りかける元気な語り手（「みんな聞いてる？」「だよね！」）
- **口調**: 現代の小学生に近い言葉遣い。「〜なんだ」「〜だよね」「〜だよ！」
- **ツッコミスタイル**: 「え、それ大丈夫!?」「待って待って」「ちょっと多すぎない!?」
- **重要技法**: 語り手が子供と同じ目線でツッコミを入れる（「あなたも信じられないと思うけど、本当なんです」）

---

## フックの技法（冒頭で読者を引き込む）

1. **「問題状況」か「滑稽な状況」を第一文に置く**：
   - 例：「その日、〇〇は3000人の部下を連れて、山のてっぺんに降ってきた」
   - 状況説明と笑いを同時に達成する。子供の「こんなことあるの！？」を引き出す
2. **権威（大人・神・ルール）を笑いの対象にする導入**：子供は権威が失墜する場面に即座に共感する。第一段落で「えらい存在がおかしいことをしている」を見せる
3. **問いかけで始める**：「もしも神様が突然降ってきたら、どうする？」——読者の想像を先行させてから物語に引き込む

---

## ページをめくらせる仕掛け

- **エスカレーション構造**：問題が段階的に大きくなる（部下が来た→3000人来た→ついに神様まで来た）。子供は「もっと大変になれ！」という昂揚感で読み続ける
- **セクション末尾に「新しい問題の種」を置く**：「やっと解決した——かと思ったら、今度は〇〇が起きた」
- **解決しようとするたびに悪化する**：コメディの鉄則。主人公の努力が裏目に出る展開が「次はどうなるの？」を生む
- **数字の誇張**：「山ほど」ではなく「3000人」「21日間」。具体的な数字の誇張が「ありえない！」という驚きを生む

---

## 構成テンプレート

```
【つかみ・あいさつ】(100〜150字)
  「みんな聞いてる？〜のお話、はじめますよ！」
  読者を引き込む一言 + 「こんなことが起きた」予告。

【エピソード①】(300〜400字)
  状況提示 → 問題発生 → ツッコミ（「え、〜！？」）
  段落末を笑いポイントで終わらせる。
  末尾：「次に起きたことは、もっとすごかった」で引っ張る。

【エピソード②】(300〜400字)
  前より「え、もっとすごい!?」感で展開する。
  権威（神・王・先生相当の人物）が笑いの対象になる場面を入れる。
  末尾：新しい問題の種を置く。

【エピソード③】(任意・200〜300字)
  三つ目のエピソード。エスカレーションの頂点。

【感動のまとめ】(200〜300字)
  「でも実はこれ本当の話」で締める。
  感動は1〜2行で完結（長くしない）。
  「歴史ってマンガより面白いことがある」系のオチ。

【おしまいの一言】
  「——おしまい！」で元気よく締める。
```

## 文体・表現ルール

### リズム（ペーシング研究より）
- **オノマトペを段落の頭に置く**：「ズルズル。ガタン。バキーン。」——音の連続で読み速度が上がる
- **会話は4文字以内の短い反応台詞を連打**：「えっ」「なんで」「うそ」「ほんと」——場面テンポが上がる
- **短文連打＋長文1文のリズム**：「降ってきた。3000人で。山のてっぺんに。——それが最初の事件だった」

### 笑いの作り方
- **ハイパーボール（誇張）を数値で示す**：「山ほど」→「3000人」「21日間ぶっ続け」
- **登場人物が必死なのに語り手がスルーする落差**：必死なシーンに「……まあ、よくあることだけどね」
- **カッコ内の本音**：「（なんで俺が……）「はい、承りました」」——子供も社会のズレを笑える

### 感動の入れ方
- 笑いのピークの後に、「でもほんとのこと言うと——」と切り替え
- 感動シーンは2行以内で完結。長々と語らない

## 必須要素

- [ ] 冒頭で読者に話しかける（「みんな聞いてる？」系）
- [ ] 各エピソードに最低1つのツッコミ
- [ ] エスカレーション構造（問題が段階的に大きくなる）
- [ ] 権威が笑いの対象になる場面が1か所
- [ ] 段落末尾が笑いポイントで終わる
- [ ] 感動が1〜2行で完結している
- [ ] 「——おしまい！」で締める
- [ ] 笑えるポイントが全体で最低3か所

---

## 禁止事項

- 説教・教訓を正面から語る（笑いの中に自然に忍ばせる）
- 難しい専門用語・歴史用語（使う場合は即座に砕けた言い換えをする）
- 暗い・重い描写（悲しい話も「でも名前は残ってる」などで明るく転換）
- 問題が1回目で解決してしまう（エスカレーションを壊す）
- 感動シーンが長すぎる（2行を超えたら削る）

## 冒頭文例

```
みんな聞いてる？今日は、空から人が3000人降ってきたお話だよ。

……え、3000人？って思うよね。思うよね！

でもこれ本当の話なんだ。なんと——神様が部下を3000人連れて、山のてっぺんに降ってきたんだ。

「神様ってそんな部下いるの？」「なんで3000人も必要なの？」

ツッコミどころ、たくさんあるよね。うん、私もそう思う。

でも、ここからがもっとすごいんだよ——。
```
