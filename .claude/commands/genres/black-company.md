# ブラック企業系 コンテンツ作成手順書

## 基本情報

| 項目 | 内容 |
|------|------|
| ジャンルID | `black-company` |
| ファイル名 | `black-company.md` |
| 対象読者 | 社会人全般（20〜50代） |
| 文字数目標 | 2500〜4000字 |
| 語り口 | 三人称・ドライなナレーター |

---

## ジャンル定義・コンセプト

歴史上の出来事を「ブラック企業あるある」として描き、笑いながら「それでも折れなかった人間の強さ」を伝える。
社会あるあるコメディよりギャグ要素が強く、不条理・理不尽の度合いが高い。
「これはブラック企業だな……」というセリフを登場人物に言わせることで自己言及的なユーモアを作る。
最後だけは「笑いから感動へ」転換し、「ブラック企業で折れなかった人間の名前は残る」で締める。

---

## 語り手の設定

- **視点**: ドライで冷静なビジネスナレーター。感情移入しない
- **口調**: ビジネス文書と砕けた口語のコントラスト
- **ブラック描写スタンス**: 「これはひどい」とは言わず、淡々と事実を並べる
- **感情の置き所**: 最後の1〜2段落のみ静かに感動を入れる

---

## 構成テンプレート

```
【入社・書類フェーズ】(300〜400字)
  「〔会社名 / 設立年〕」の見出し
  就労条件が異常なことを淡々と書類形式で提示
  「……残業代は？」「ありません」のやり取り

【エピソード①「〜あるある」】(400〜500字)
  最初の試練・不条理なルール
  社内チャット・勤怠システム・研修など現代フォーマット
  「（登場人物）「これブラック企業じゃないですか」
  上司「ブラックだから」」のようなメタな一言を入れる

【エピソード②〜④】(各400〜500字)
  異なる歴史人物の不条理エピソード
  それぞれ異なるフォーマット（週報・評価面談・業務連絡など）
  各エピソード末尾に「*ブラック企業の真実：〜*」または「*最強のメンタルとは：〜*」

【まとめ・退職者アンケート】(200〜300字)
  「退職者アンケート」Q&A形式
  各人物の回答→最後の一人だけ「——（回答なし）」で余韻

【締め】(100〜200字)
  ブラック企業で折れなかった人間の名前が今も語り継がれることを一段落で
  「——お疲れ様でした。」で締める
```

---

## ブラックコメディの作り方

### 不条理の三段階
1. **設定の不条理**: 就労条件・規則自体がおかしい（「休日未定」「残業代なし」）
2. **反応の不条理**: 登場人物が理不尽に慣れすぎている（「まあブラックだから」）
3. **結果の不条理**: 結果がすごすぎて褒める気も引けない（「全勝した人を謹慎？」）

### メタなギャグの入れ方
- 登場人物が「これブラック企業だな」と自己言及する
- 現代の常識（「カウンセリング制度は？」「コンプライアンス違反では？」）で過去を突っ込む
- ナレーターが「これが5000年前から変わらない職場の真実である」と締める

### ギャグと感動の切り替え方
- ギャグのテンションをキープしたまま、最終段落だけトーンを落とす
- 「笑い話にしてはいけない部分もある」という一行で転換を宣言する
- 感動は1〜3行で完結させる（長くしない）

---

## 使用フォーマット一覧（social-comedyと重複可だが濃度を上げる）

| フォーマット | ブラック的な使い方 |
|------------|----------------|
| 就労条件書 | 明らかにおかしい条件を「普通に」列挙 |
| 社内チャット | 深夜・緊急・音信不通のやり取り |
| 勤怠管理 | 「休日未定」「有給消化率0%」 |
| 業績評価メール | 全勝者でも表彰後即謹慎の不条理 |
| 退職理由 | 「天命を全うしたので」「戦死のため」 |
| コンプライアンス違反報告 | 存在するが機能しない制度 |

- [ ] 「——お疲れ様でした。」で締める

---

## 禁止事項

- 感動シーンが長すぎる（1〜3行に厳守）
- 「本当は大変だった」と説明的に書く（笑いの中で伝わるようにする）
- social-comedyと同じフォーマット・同じ「あるある」の焼き直し
- ブラック労働の美化・礼賛（あくまで「それでも折れなかった」が主役）

```
- 勤務地：（場所）（交通手段なし）
- 勤務時間：24時間365日（休日未定）

「……残業代は？」

「そうですか」

*（役職）もサービス残業。これが株式会社・○○のリアルだ。*
```
