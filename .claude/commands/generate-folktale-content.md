# /generate-folktale-content

PDFから抽出したテキストをもとに、**日本昔話風**のMDファイルを生成するスキル。

## 使い方

```
/generate-folktale-content <slug> <本のタイトル>
```

例：
```
/generate-folktale-content korean-epic 韓民族選民大叙事詩
```

実行すると：
1. 下記の作成方針を表示してユーザーに確認を求める
2. 確認後、`content/books/<slug>/folktale.md` を生成する

---

## 日本昔話風 作成方針

### コンセプト

年齢・学歴を問わず、**語り聞かせ**の形式で本の核心を伝える。
「むかしむかし〜じゃった」の語り口で、神話・歴史・思想を
日本の昔話の文法に乗せて再構成する。

---

### 文体・語彙

| 要素 | 指針 |
|------|------|
| 書き出し | 「むかし、むかし、〜」または「遠い遠い昔のこと、〜」で始める |
| 語り口調 | 「〜じゃったとさ」「〜けり」「〜であったそうな」「〜とのことじゃ」 |
| 登場人物 | 神・王・侍・老翁・娘など昔話的キャラクターで擬人化・置換 |
| 比喩 | 抽象概念を自然・動植物・天気で表現（選民の使命 → 山の頂に宿る火 等） |
| テンポ | 三段構成（困難 → 試練 → 解決）の昔話の型を踏む |
| 結び | 「〜とさ。めでたし、めでたし。」または余韻を残す終わり方 |

---

### 構成パターン

```
[語り出し]
むかし、むかし、大きな海のむこう、白い山のそびえる国に……

[世界観の設定]
その国の民は、空の神さまを「天の父母さま」と呼んで、
毎朝、手を合わせ、夜には星を眺めながら……

[中心となる試練や使命]
ある日、天の父母さまはひとりの老翁をお呼びになってこう申された。
「そなたの民に、大切な使命を授けよう……」

[クライマックス]
（具体的な歴史的エピソードを昔話のクライマックスとして描く）

[結び・教訓]
「民が心をひとつにするとき、天は必ず味方をする」
——そう語り継がれておる。
```

---

### 内容方針

- **本の宗教的主張は「神話・伝説」の文脈で自然に包む**
  宗教的に敏感な主張を直接主張せず、「〜という伝えがある」「〜と語られてきた」という形で語り継ぎの形式に包む

- **歴史的事実は昔話のエピソードとして活かす**
  ハングル発明 → 「王さまが民のために文字を作った話」
  李舜臣 → 「海の将軍と亀の船の話」
  柳寛順 → 「少女が万歳を叫んだ話」

- **日本の昔話との橋渡しを意識する**
  日本読者が「浦島太郎」「桃太郎」を連想しやすい比喩・展開を用い、
  韓国の物語を親しみやすく伝える

- **全年齢が読めるが、大人にも味わいがあること**
  表面は子ども向けの語り口でも、深読みすると思想・歴史の層が見えるように

### フロントマター

```yaml
---
title: 〈昔話風タイトル〉
level: folktale
style: 日本昔話風
---
```

### 分量

1500〜2500字（長くなりすぎず、語り聞かせで15〜20分程度）
