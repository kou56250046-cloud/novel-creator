# 都市伝説系 コンテンツ作成手順書

## 基本情報

| 項目 | 内容 |
|------|------|
| ジャンルID | `urban-legend` |
| ファイル名 | `urban-legend.md` |
| 対象読者 | 若年層〜社会人（15〜40代） |
| 文字数目標 | 2500〜4000字 |
| 語り口 | 一人称・匿名の調査者/研究者 |

---

## ジャンル定義・コンセプト

歴史上の出来事を「公式記録には載らなかった側面」として再解釈し、都市伝説・陰謀論のトーンで語る。
読者が「これは本当かもしれない」と思わせる曖昧さと、「作り話かもしれない」という余地の両方を残す。
各エピソードの末尾に「封印レベル」評価を入れることで、ドキュメンタリー感とゲーム感を両立する。
最後は「調べ続けるかどうかはあなた次第」という形で読者に委ねる。

---

## 語り手の設定

- **正体**: 名前も立場も明かさない匿名の調査者。「ある資料館の関係者」「元研究者」程度
- **口調**: 落ち着いている。感情的にならない。ただし末尾で少しだけ警戒を見せる
- **信頼性**: 高くも低くもない。「確認できることとできないことを分けて書く」スタンス
- **関係性**: 読者に直接語りかける（「あなたは」「あなたが」）

---

## 構成テンプレート

```
【はじめに】(200〜300字)
  「この記録を読んでいるあなたに確認しておきたいことがある」
  なぜこれを書いているか、どういう経緯でこの情報にたどり着いたか
  「公開するのはリスクがある」という一文で引き込む

【第一の記録〜第N（各エピソード）】(各400〜600字)
  見出し：「第〇の記録｜〇〇」
  ①公式の歴史的事実を提示（教科書レベル）
  ②「ただし——」で転換。矛盾点・説明できない点・消された記録を提示
  ③証拠・証言・記録の欠落を積み上げる
  ④結論は出さない。「誰が調べたか、なぜ続きが書かれなかったか」で余韻
  末尾：「*封印レベル：低/中/高/最高。〇〇。*」

【おわりに】(200〜300字)
  「あなたは今、一つの選択の前に立っている」
  調べ続けるか、閉じるか——どちらも正しいと提示
  最後に「この記録が削除される前に転送してほしい」などの一行で締める
```

---

## 封印レベルの書き方

各エピソードの末尾にイタリックで付ける。レベルに応じてコメントを変える。

| レベル | 例 |
|--------|-----|
| 低 | `*封印レベル：低。しかし調べ始めると止まらなくなる。*` |
| 中 | `*封印レベル：中。〇〇に特別な許可が必要な頁がある。*` |
| 高 | `*封印レベル：高。これについて質問すると答えが毎回微妙に違う。*` |
| 最高 | `*封印レベル：最高。この項目を書いた直後、私はこの文書をバックアップした。*` |

## 「信頼できる疑問」の作り方

### やること
- 本当に存在する歴史的事実をベースにする
- 「記録が残っていない」「論文が撤回された」「査読を通らなかった」など実際にあり得る出来事を使う
- 数字・固有名詞・年号を入れてリアリティを出す
- 結論は絶対に出さない（「〜だったのかもしれない」止まり）

### やらないこと
- 特定の民族・宗教・存命人物を直接批判する陰謀論
- 「宇宙人」「超能力」など非現実要素を断言する
- オカルト系の過剰な演出（呪いが〜、霊が〜）
- すべてを説明しすぎる（余白を残す）

---

## 文体・表現ルール

### 引き込みの技法
- 段落を短くする（2〜3行で改行）
- 「ただし——」「問題はそこではない」「ここで少し立ち止まってほしい」で読者を引き止める
- 疑問文で段落を終える（「では、なぜ消えたのか」）

### リアリティの出し方
- 「2003年に短い論文を書いた」「2年後に撤回した」など具体的な年号
- 「日朝両方の一次資料に残っている」など複数の情報源を匂わせる
- 「現在は〜に登録されている。閲覧に許可が必要な頁がある」など実在する仕組みを使う

### 余韻の作り方
- エピソードの結論は「〜は不明だ」「調べた人間がいたかもしれない」で終える
- 「おわりに」は説明しない。読者の想像に委ねる
- 最終行は短く（「念のため。」「転送してほしい。」）

---

## 必須要素

- [ ] 冒頭に「あなたに確認しておきたい」系の語りかけ
- [ ] 歴史的エピソードが最低3つ
- [ ] 各エピソードに「公式の事実 → 矛盾点 → 消えた記録」の流れ
- [ ] 各エピソード末尾に「*封印レベル：〜*」
- [ ] 結論を出さない（余白を残す）
- [ ] 「おわりに」で読者に選択を委ねる
- [ ] 最終行に短い余韻の一文

---

## 禁止事項

- 「実はこういう真実だった」と断言する
- 特定の国・民族・思想への批判を目的とした内容
- 根拠なく存命の実在人物を巻き込む
- ホラー・オカルトの演出に頼りすぎる（都市伝説はあくまで「かもしれない」）
- 過度に長い説明（テンポを殺す）

## 冒頭文例

```
## はじめに

この文書を読んでいるあなたに、一つ確認しておきたいことがある。

（検索エンジン名）で「（キーワード）」と検索すると、ある時点から検索結果が変わる。
以前はヒットしていた（資料の種類）が——消えている。

これから書くことは、その（資料名）の内容の一部だ。

書いた人間の名前は出さない。本人が希望していないし、今は消息も不明だ。
```
