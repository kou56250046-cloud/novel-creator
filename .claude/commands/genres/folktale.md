## ジャンル定義・コンセプト

日本の昔話（桃太郎・浦島太郎など）の語り口を借りて、原典の内容を温かい語りで伝える。
難しい概念を排除し、繰り返しのリズムと情景描写で子供が聞いても楽しめる形にする。
「めでたし、めでたし」で締めくくることで、読後感を安定させる。

**核心原則（プロップ＋McKee）：**
「欠如→出発→試練→援助者→解決→帰還→報酬」の7段階構造を骨格にする。
困難は必ず3回繰り返し（ルール・オブ・スリー）、3回目に解決する。

---

## 語り手の設定

- **視点**: 全知の語り手（「〜じゃったとさ」「〜おったんじゃな」）
- **口調**: 方言風（〜じゃ、〜おった、〜とさ、〜のう）
- **感情**: 温かく、穏やか。驚きも「おや」「まあ」程度
- **重要技法**: 語り手が感情コメントをする（「なんと、それは大変なことじゃったのう」）。これが読者の感情反応に「許可」を与える

---

## フックの技法（冒頭で読者を引き込む）

1. **儀式的開幕フォーミュラ**：「むかし、むかし」で始める。聴衆の脳内に「物語モード」スイッチを入れる口承文学の技術。第一文に「場所+人物+性格」の3要素を同時に盛り込む
   - 例：「むかし、むかし、白い山のある国に、心のやさしい男の子がおったとさ」
2. **不均衡な状態から開始**：主人公が何かを「持っていない」か「失った」状態から始める。これが読者に「どうなるの？」という問いを植える（Truby原則）
3. **伏線としての「不思議な出会い」を導入末尾に置く**：「ところがその日、山道で奇妙なじいさまに会ったとさ」——これが謎として機能し、次のセクションへ引っ張る

---

## ページをめくらせる仕掛け

- **リフレインで読者を参加させる**：「〜とさ、〜とさ」のように繰り返すフレーズは、幼い聴衆が声に出して一緒に言えるようにする。これが「やめられない」状態を生む
- **試練の二択を見せてから選ばせる**：「助けを借りる道」と「一人で解決する道」を提示してから主人公に選ばせる。読者は予測して、次ページで答え合わせをしたくなる
- **オープニングイメージ＝クロージングイメージ原則**：冒頭と結末に同じ情景（山・川・空）を置く。変化した点が主人公の成長を証明し、読者に達成感を与える

## 構成テンプレート

```
【導入】(200〜300字)
  むかしむかし、〜の国に〜がおった。
  「場所 + 主人公 + 欠如（持っていないもの）」を一段落に収める。
  末尾に「不思議な出会い」の予兆を置く。

【試練①（失敗）】(300〜400字)
  最初の試みは失敗する。
  「ところが——」で転換。
  末尾：「〜はうなだれたとさ」で読者の共感を引き出す。

【試練②（失敗）】(300〜400字)
  二度目も失敗。前回と別の方法で試みる。
  擬音語・擬態語を1段落に最低1つ入れる。

【試練③（成功）】(400〜500字)
  三度目に成功。援助者（動物・老人・神など）の助けで乗り越える。
  「ドスン」「フワリ」など音で場面を作る。
  教訓となる行動・価値観を「行動の結果」で示す（説明しない）。

【結び】(200〜300字)
  「こうして〜」でまとめ。
  冒頭と同じ情景描写で締める（成長を暗示）。
  「——めでたし、めでたし。」で閉じる。
```

## 文体・表現ルール

### 語尾・語り口
- 文末は「〜じゃった」「〜おった」「〜とさ」「〜んじゃな」を使う
- 現代語の「〜です」「〜ます」は使わない
- 語り手の感情コメントを各セクションに1つ入れる：「なんと、〜」「おや、まあ〜」

### リズム（ペーシング研究より）
- **短文×3連打 → 長文×1**：緊張場面で「斬る。受ける。転ぶ。——そのとき、大きな声が響いたとさ」
- **繰り返しフレーズ**：同じ言葉を2〜3回繰り返す。幼い読者の記憶に刻む
- **擬音語・擬態語**：「ドスン」「フワリ」「キラキラ」を各段落に1つ

### 教訓の埋め込み方
- 「正直が大事」とは書かない。「正直に言ったら報われた」という結果だけを見せる
- セリフで語り手に「なんと〜」と驚かせることで、読者の感情反応を誘導する

## 必須要素

- [ ] 「むかし、むかし」で開始
- [ ] 主人公に「欠如（持っていないもの）」がある
- [ ] 試練が3回（ルール・オブ・スリー）
- [ ] 末尾に「伏線となる予兆」または自然描写で情景を締める
- [ ] 擬音語・擬態語が最低3か所
- [ ] 語り手のコメント（「なんと〜」）が最低2か所
- [ ] 「——めでたし、めでたし。」で締める

---

## 禁止事項

- 難解な歴史用語・専門用語をそのまま使う
- 暗い・重い結末（悲劇は「でも民は覚えておった」などで昇華する）
- 教訓を直接書く（「〜が大事」「〜を学んだ」）
- 現代語・カタカナ語（「システム」「プロジェクト」など）
- 試練が1回で成功してしまう（必ず2回失敗させる）

## 冒頭文例

```
むかし、むかし、大きな海のむこう、白い山のそびえる国のことじゃった。

その国の民は、雪のように白い服をこよなく愛しておった。
朝は天に手を合わせ、夜は星を眺めながら歌い、笑い、泣いて暮らしておったとさ。

ところがある日——山の頂に、見たこともない光が降りてきたとさ。
「いったい、なんじゃろう……」と村人たちは、首をかしげたとさ。
```
