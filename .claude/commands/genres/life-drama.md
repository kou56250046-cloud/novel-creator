# 人生ドラマ コンテンツ作成手順書

## 基本情報

| 項目 | 内容 |
|------|------|
| ジャンルID | `life-drama` |
| ファイル名 | `life-drama.md` |
| 対象読者 | 中年層（40〜60代） |
| 文字数目標 | 3000〜5000字 |
| 語り口 | 一人称・語り手は50代前後の人物 |

---

## ジャンル定義・コンセプト

人生の折り返し地点に立つ語り手が、歴史上の人物の言葉・行動を手がかりに「自分の人生の意味」を静かに問い直すエッセイ調の物語。
「人生を問い直す問い」「歴史人物との対話」「過去の後悔と和解」の三層構造で感情を積み上げる。
説教・自己啓発にならないよう、答えは出さずに「余韻で終わる」ことが最重要。

---

## 語り手の設定

- **年齢**: 50〜55歳（定年が見えてきた時期）
- **状況**: 仕事・家族・自分の価値に関して漠然とした問いを抱えている
- **きっかけ**: 歴史書・親の遺品・病院の待合室など静かな場所で歴史に触れる
- **口調**: 知的・内省的・穏やか。自分に問いかけながら語る

---

## 構成テンプレート

```
【序】(200〜300字)
  語り手がいつ・どこで・なぜ歴史書を開いたか
  「自分はいったい何のために生きてきたのか」という問いの提示
  （唐突に提示せず、日常の一場面から自然に滲み出る形で）

【一〜N（各歴史人物の章）】(各400〜600字)
  歴史人物のエピソードを引用
  → 語り手が自分の半生と重ねる
  → 「私はどうだったか」という自問
  → 答えは出さない。問いが深まるだけ

【結び】(300〜400字)
  すべての歴史人物を経て、語り手が今感じることを静かに語る
  「それで十分だ」「それだけでいい」系の穏やかな着地
  答えを出さず、「問いと共に生きる」ことを受け入れる
  原典の核心的な言葉で締める
```

---

## 文体・表現ルール

### エッセイ調の書き方
- 語り手が過去を振り返るとき：「あのとき、私は〜した」「〜年前のことだ」
- 歴史を引用するとき：「記録にはこうある。〜」「〜と彼は言った。」
- 自問するとき：「私はどうだったか。」（短い文で行を分けて問い）

### 感情の抑え方
- 感情的な語彙を避ける（「悲しい」ではなく「胸が重くなった」）
- 泣く描写は「目が熱くなった」程度に留める
- 感動的な場面ほど文を短くして余白を作る

### 歴史との対話
- 歴史人物の言葉を「引用」する形で提示する（「〜と、記録に残っている」）
- 語り手がその言葉に「返答」するように自分の経験を語る
- 明示的な「このような教訓がある」は禁止。経験で語らせる

### 結びの作り方
- 「答え」を出さない（「わからない」も「わかった」も正解）
- 「問いと共に生きる」「それで十分だ」という静かな受け入れ
- 小さな行動（茶を淹れる・窓を開けるなど）で締める
- 朝の光・川の音など情景描写で余韻を作る

---

## 必須要素

- [ ] 語り手の「人生の問い」が冒頭で提示される
- [ ] 歴史人物のエピソードが最低3人分
- [ ] 各エピソード後に語り手の「自問」
- [ ] 答えを出さない結び
- [ ] 原典の核心的な言葉・価値観が自然に登場
- [ ] 全体を通して説教・自己啓発にならない

---

## 禁止事項

- 「この話から学べることは〜」という直接的な教訓提示
- 語り手が急に明るくなる・問題が解決する（穏やかな余韻を優先）
- 感情的な泣き崩れ・叫び（静かな感情表現に留める）
- 自己啓発書風のまとめ（「5つの教訓」など）
- 若者向けの言葉遣い・流行語

## 冒頭文例

```
私が（歴史書・本）を初めて開いたのは、（年齢・具体的な状況）だった。

（日常の一場面を具体的に描写：定年まであと〜年・子供が独立した後・病院の待合室など）

「自分はいったい、何のために生きてきたのだろう」

問いは唐突に訪れた。

それは（場所）で、ふと（状況）した瞬間のことだった。
```
